# Django Docker Practice Project

Goals:
- Build a 1-tier Django app
- Create a multi-stage Dockerfile
- Convert it to a distroless image
- Add Gunicorn
- Add healthchecks

Tasks:
1. Create Django project and app.
2. Create Note model.
3. Create CRUD APIs.
4. Build image.
5. Add multi-stage build.
6. Convert runtime to distroless.
7. Add healthcheck endpoint.
